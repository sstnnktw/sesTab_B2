การสำรวจภาวะเศรษฐกิจและสังคมของครัวเรือน
ตอนที่ 1 สมาชิกของครัวเรือน
ข้อที่มี * หมายถึง ต้องมีการบันทึกข้อมูล ปล่อย blank ไม่ได้
* HM01N1 : คำนำหน้าชื่อ
บันทึกเป็นตัวแปร HM01N1_T


ถ้า HM01N1 = 1 - 5 → เพศ(HM03) เปลี่ยนตามคำนำหน้าชื่อ (auto)
ถ้า HM01N1 = 6 → แสดงตัวเลือกของ HM03 ให้เลือก
* HM01N2 : ชื่อ
* HM01N3 : นามสกุล
* HM02 : ความสัมพันธ์กับหัวหน้าครัวเรือน


ถ้า HM01 = 1 → HM02 = 1 (แก้ไขไม่ได้)
ถ้า HM01 != 1 → แสดงตัวเลือก 2-13, 00
* HM03 : เพศ


ถ้า HM01_1N = 1 - 6 → เพศ(HM03) เปลี่ยนตามคำนำหน้าชื่อ (auto)
ถ้า HM01_1N = 6 → แสดงตัวเลือกของ HM03 ให้เลือก
* HM04 : อายุนับถึงวันเกิดครั้งสุดท้าย
เด็กอายุต่ำกว่า 1 ปี บันทึก "000"
กรณีที่อายุมากกว่า 120 ปี บันทึก "120"
* HM05 : ศาสนา
บันทึกเป็นตัวแปร HM05_T
* HM06 : ภาษาส่วนใหญ่ที่ใช้คุยกับสมาชิกในครัวเรือน
บันทึกเป็นตัวแปร HM06_T
* HM07 : สัญชาติ
* HM08 : เป็นผู้พิการหรือไม่
* HM09 : สามารถดูแลตนเอง (ในชีวิตประจำวัน) โดยไม่ต้องมีคนช่วยเหลือได้หรือไม่
* HM10 : สถานภาพสมรส
ถามเฉพาะสมาชิกที่มีอายุ 15 ปีขึ้นไป
* HM11 : มีพ่อแม่อาศัยอยู่ด้วยหรือไม่
ถามเฉพาะสมาชิกที่มีอายุต่ำกว่า 15 ปี
บันทึกเป็นตัวแปร HM11_T
* HM12 : จำนวนปีที่ไม่ได้อยู่กับพ่อ และ/หรือ แม่
ถามเฉพาะสมาชิกที่มีอายุต่ำกว่า 15 ปี
* HM13 : สาเหตุที่ไม่ได้อยู่กับพ่อ และ/หรือ แม่
ถามเฉพาะสมาชิกที่มีอายุต่ำกว่า 15 ปี
บันทึกเป็นตัวแปร HM13_T
* HM14 : ขณะนี้กำลังเรียนหนังสืออยู่หรือไม่
ถ้า HM14 = 0 → ข้ามไปถาม HM22
* HM15 : ชั้นสูงสุดที่เรียนจบ
* HM16 : สาขาของ HM15 (ถ้ามี)
(ถ้าชั้นสูงสุดที่เรียนจบเป็นอาชีวะและปริญญาให้ระบุสาขาด้วย)
(อนุปริญญา ไม่บันทึกสาขา)
* HM17 : ชั้นที่กำลังเรียน
ถามเฉพาะสมาชิกที่ HM14 = 1 หรือ 2
ค่าใช้จ่ายด้านการศึกษา (ตลอดปีการศึกษาที่ผ่านมา)
ข้อไหนไม่มีค่าใช้จ่าย ปล่อย blank ได้ (แต่หลังบ้านบันทึกค่าเป็น 0)
(ถ้าไม่ได้จ่ายเองให้ประเมินมูลค่า ยกเว้นนโยบายเรียนฟรี 15 ปี ไม่ต้องประเมิน)
HM18 : ค่าเล่าเรียน/ค่าธรรมเนียมการศึกษา/ค่าบำรุงการศึกษา
HM19 : ค่าเครื่องแบบ (ทุกประเภทที่กำหนด)
HM20 : ค่าสมุดหนังสือ/อุปกรณ์การเรียนอื่น ๆ
HM21 : ค่าเดินทางไปเรียน
(ปัจจุบัน) มีสวัสดิการสวัสดิการด้านการรักษาพยาบาลและ/หรือสวัสดิการอื่น ๆ ต่อไปนี้หรือไม่
* HM22 : สิทธิเบิกค่ารักษาพยาบาลจากหน่วยงานราชการ/รัฐวิสาหกิจ
* HM23 : สิทธิประกันสุขภาพถ้วนหน้า (บัตรทอง)
* HM24 : สิทธิประกันสังคม ม.33, 39 (ลูกจ้าง/ผู้ประกันตน)
* HM25 : สิทธิประกันสังคม ม.40 (ผู้ประกอบอาชีพอิสระหรือแรงงานนอกระบบ)
* HM26 : ประกันสุขภาพเอกชน
* HM27 : สวัสดิการค่ารักษาพยาบาลจัดโดยนายจ้าง
* HM28 : สวัสดิการค่ารักษาพยาบาลอื่น ๆ เช่น สวัสดิการครูเอกชน สวัสดิการค่ารักษาของนักเรียน ฯลฯ (ระบุ)
บันทึกเป็นตัวแปร HM28_T
(ในรอบ 12 เดือนที่แล้ว) ได้รับผลประโยชน์ (ไม่ต้องคืน) จากโครงการช่วยเหลือต่อไปนี้หรือไม่
dropdown ต้องเลือกคำตอบทุกข้อ แต่จำนวนสามารถปล่อย blank ได้ (หลังบ้านบันทึกค่าเป็น blank)
* HM29 : เงินสงเคราะห์เพื่อการยังชีพสำหรับผู้สูงอายุ

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_1 (เก็บไว้เป็น default ของ IN08_1 ใน สศส.3)
* HM30 : เงินสงเคราะห์สำหรับผู้พิการ

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_2 (เก็บไว้เป็น default ของ IN08_2 ใน สศส.3)
* HM31 : อาหารกลางวัน/อาหารเสริมฟรี (นม) สำหรับนักเรียน

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_3 (เก็บไว้เป็น default ของ IN08_3 ใน สศส.3)
* HM32 : ทุนการศึกษาจากรัฐ

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN09 (เก็บไว้เป็น default ของ IN09 ใน สศส.3)
* HM33 : บัตรสวัสดิการแห่งรัฐ (ผู้มีรายได้น้อย)

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_4 (เก็บไว้เป็น default ของ IN08_4 ใน สศส.3)
* HM34 : เงินอุดหนุนเพื่อการเลี้ยงดูเด็กแรกเกิดจนถึงอายุ 6 ปี (พม.)

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_5 (เก็บไว้เป็น default ของ IN08_5 ใน สศส.3)
* HM35 : เงินสงเคราะห์บุตรแรกเกิดจนถึงอายุ 6 ปี (ประกันสังคม)

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN15 (เก็บไว้เป็น default ของ IN15 ใน สศส.3)
* HM36 : โครงการช่วยเหลือของรัฐอื่น ๆ (เช่น โครงการเราเที่ยวด้วยกัน โครงการช่วยเหลือเกษตรกร ฯลฯ)

จำนวนเงิน/มูลค่าที่ได้รับ :


บันทึกเป็นตัวแปร HM_IN08_6 (เก็บไว้เป็น default ของ IN08_6 ใน สศส.3)
(ปัจจุบัน) ได้กู้ยืมเงินจากกองทุนต่าง ๆ (ต้องใช้คืน) ที่รัฐจัดให้ต่อไปนี้หรือไม่
คำว่า “ปัจจุบัน” หมายถึง การที่บุคคลยังมีหนี้สินที่ยังค้างชำระอยู่ของโครงการนั้น ๆ
* HM38 : โครงการเงินกู้เพื่อการศึกษาของรัฐ
* HM39 : โครงการธนาคารประชาชนเพื่อผู้ประกอบอาชีพอิสระรายย่อย
* HM40 : กองทุนหมู่บ้าน/กองทุนชุมชนเมือง
* HM41 : กองทุนอื่น ๆ (ระบุ)

อาชีพหลัก
* HM42 : อาชีพหลักหรืองานที่ใช้เวลาทำส่วนใหญ่ (ในรอบ 12 เดือนที่แล้ว)
ถ้าไม่ได้ทำงานหารายได้ ให้ระบุสาเหตุด้วย เช่น
แม่บ้าน/ทำงานบ้าน = 9991                                                         เรียนหนังสือ(อายุ 15 ปี ขึ้นไป) = 9992
เด็ก(อายุต่ำกว่า 15 ปี)/ คนชรา(อายุ 60 ปีขึ้นไป) = 9993                ป่วย/พิการจนไม่สามารถทำงานได้ = 9994
กำลังหางานทำ = 9995                                                               ไม่สมัครใจทำงาน = 9999
อื่น ๆ เช่น เกษียณอายุ = 9996


If HM42 = 9991 → HM43 = 08 → จบตอนที่ 1
If HM42 = 9992 → HM43 = 09 → จบตอนที่ 1
If HM42 = 9993 → HM43 = 10 → จบตอนที่ 1
If HM42 = 9994 → HM43 = 11 → จบตอนที่ 1
If HM42 = 9995 → HM43 = 12 → จบตอนที่ 1
If HM42 = 9999 → HM43 = 13 → จบตอนที่ 1
If HM42 = 9996 → HM43 = 14 → จบตอนที่ 1
